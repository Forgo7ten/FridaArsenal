# FridaArsenal

Frida武器库，集成常用hook脚本与代码模板，自用。

-   下载

    ```bash
    git clone --recurse-submodules https://github.com/Forgo7ten/FridaArsenal
    ```

-   或者

    ```bash
    git clone https://github.com/Forgo7ten/FridaArsenal
    git submodule init
    git submodule update
    ```

-   使用

    ```
    npm install
    npm run watch
    frida -UFl _agent.js
    ```
